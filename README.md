# To Do List - JS
PWA creada para tener un registro de las actividades que los usuarios deben realizar. 
Se desarrolló con JavaScript y Firebase.
## Funcionalidades:
- Crear nota
- Titulo
- Descripción
- Añadir fecha límite si es necesario

## Interfaz:
  - Se muestran las tareas
  - Eliminar tareas
  - Ver detalle
